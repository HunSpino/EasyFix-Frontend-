## Setup
A konzolba írd be eme parancsot az előkészülethez

```
npm install
```

Majd ha minden jól ment, indítsd el a fejlesztői szervert

```
npm run serve
```

Emlékeztető: A Backend-nek funia kell ahhoz hohy a frontend működjön
